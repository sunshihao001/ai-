# 05 Maturity L0-L5

## 分级
- L0：原始碎片
- L1：可理解观点
- L2：局部理论
- L3：端口模型
- L4：知识循环结构
- L5：可复用内容工程模块

## 判定模板
- 端口：A / B / C / A-B / B-C / A-B-C
- 成熟度：L0-L5
- 状态：raw / candidate / accepted / partial / needs_evidence / needs_full_text / rejected
- 推荐落点：具体文件
- 下一步动作：澄清 / 补证据 / 合成 / 归档 / 暂停

## 当前示例
- A端是 dominant front door → L4 / accepted
- A→B→C 是常见循环 → L4 / accepted
- dbskill 是命令家族不是新底座 → L3 / accepted
- X 文章摘要方向 → L1 / needs_full_text
